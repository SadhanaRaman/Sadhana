The site purpose and idea.
The purpose of the site as mentioned earlier is to document my journey in technology. I got the idea one random evening, when I had a little too much time on my hand and thought how fun it'd be to create a website with the creative freedom to design it the way I want. Although it is a pet project, it is designed for an audience with a basic technical backgroud.
The tech ?
The site is developed from scratch without pre-built templates, code snippets or website builders. It is designed using HTML5, CSS, Javascript, JQuery and Bootstrap. Font Awesome is used for the icons. Some of the clientside validations were implemeted with Ajax and SMTP client is used for emails (you'd know if you submit your comments :)). Visual Studio Community is the IDE, VSTS is used for version control and deployed using Azure.
The theme ?
The site has an overall Azure theme, which happens to be my favourite color and coincidentaly Microsoft's deployment technology (Btw if you happen to know why Microsoft decided to name their tech Azure, let me know. Although I'm guessing it has something to do with Azure skies. A sensible name too.) Custom CSS templates were applied, such as the accordion in the Skills page (overriden to have multiple panels open simultaneously, custom icons.)
